# react-hooks
React의 Hooks 특징과 생명주기 정리

## 훅이란?
`use`로 시작하는 함수들을 Hooks라고 함.   
다른 함수들보다 제한적.  
컴포넌트의 최상단에서만 호출할 수 있음.  
조건문이나 반복문 안에서 사용하려면, 컴포넌트를 새로 만들어서 넣어야 한다.
